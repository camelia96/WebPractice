  • 생활코딩 HTML 후반강의

    • 오늘은 생활코딩 WEB시리즈중 첫번째인 WEB1을 들었다. 많은 것을 배웠기 때문에 잊지 않으려면 직접 활용해보는 것이 중요하다.

    1. 웹 페이지에 동영상 넣는법

      2. ex) 내 홈페이지에 유튜브 동영상을 넣고 싶다? 그러면 원하는 유튜브 동영상 사이트를 들어가서 동영상에 오른쪽 마우스 버튼을 누르면 동영상 소스코드 복사가 있다. 그리고 에디터에 붙여넣어주면 완성!

      여기서 주목해야할 점은 iframe 이라는 태그와 그 안에 들어가 있는 속성들이다.

    2. 웹 페이지에 댓글 기능 추가하는 법

      3. 댓글 기능을 추가하는 법은 html로는 불가능 하기 때문에 벡엔드 기술을 빌려와야 한다. 예를들면 JSP, 장고, PHP 같은 것들이다. 그리고 이것들만 필요한게 아니라, 상당한 기술력을 동반한다.
      우리가 직접 개발 할 수도 있겠지만, 지금은 시작하는 단계이기 때문에 남들이 만들어 놓은 기능을 가져오는 방법을 사용해 볼 것이다.
      DISQUS와 LIVERE라는 좋은 서비스가 있다

      ex) DISQUS를 사용할 때
      로그인을 하고 Create a new site를 찾아서 요구하는 빈칸을 작성한다. 요금에 따라 제공하는 서비스가 다르지만, 지금 간만 보는 거니까 free를 선택해준다. 그러면 본래 사용하는 서비스가 있으면 선택할 수 있지만, 지금은 없기때문에 맨 밑에 universal code를 선택한다. 그러면 코드가 나오는데, 코드의 내용은 몰라도된다. 하지만 그 코드를 복사해서 에디터에 복사하면 댓글기능 추가가 된다.

      보안상에 이유로 웹서버를 이용하지 않고, 댓글기능을 추가하면 We were unable to load Disqus 라는 말이 나오면서 댓글기능을 사용하지 못함. 그러므로 웹서버를 이용해야함.

    3. 채팅 기능 추가하는 법

      4. 댓글 기능 추가하는 법과 같이 채팅 기능을 추가하는 법은 상당히 힘든 일이다. 그렇기 때문에 이 부분도 다른 사람의 코드를 빌려오도록 하자.
      tawk를 사용해보자.
      tawk에 들어가서 로그인하고 Admin화면으로 들어가면 Widget Code가 있다. 그거 복사해서 웹사이트에 붙여넣으면 끝!

    4. 웹사이트 방문자 분석기 이용해보기

      5. 웹사이트 방문자 분석기에는 다양한 기능이있다. 예를들면 웹사이트에 몇명이나 왔는지, 어떤경로로 이 사이트에 도착했는지, 이동하는지, 어떤환경에서 웹사이트를 검색하고있는지 같은 고급정보를 알 수 있다. 여러가지 서비스가 있지만, 구글 에날리틱스 라는 서비스를 선택했고, 이 서비스의 사용법을 살펴보면서 분석기라는 서비스의 일반적인 동작방법과 사용방법을 알 수 있고, 다른 서비스를 사용할 때 참고할 수 있게 될 것이다.
      일단 홈페이지에 들어가서 계정을 만들어야 하는데, 지금 내가 만들고 있는게 웹사이트인지 앱인지 선택을 한다. 그후에 빈칸을 다 채운 후 get tracking id를 선택한다. 약관을 동의할 때 디폴트가 미국인데 대한민국으로 바꿔준다. 이렇게 다 하면 데이터를 분석할 수 있는 페이지가 뜬다. 여기서 추적 코드를 복사해서 에디터에 복사하면 구글분석기가 알아서 분석해 준다. 주의할점은 위에서 배운 코드들은 body태그에 삽입했지만, 이 코드는 head태그 안에 삽입해야한다.
