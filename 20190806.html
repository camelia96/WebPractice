  • 생활코딩 JS 시작

    1. script 태그

      2. 자바 스크립트는 일반적으로 html문서 위에서 작동된다. html위에서 자바 스크립트를 사용하기 위해선 script 라는 태그를 사용한다.
      script 태그 위에 document.write('hello world'); 라는 문구를 써보자. 그럼 웹페이지에 라는 문구가 뜬다.
      하지만 그냥 html 파일위에 문구를 쓰는것과 무엇이 다른가? 예를 들어 html에 1+1을 했을 때, 결과값으로 1+1을 보여준다. 즉, 정적이라고 볼 수 있다.
      but script 태그 위에 document.write('1+1'); 을 썼을 때 결과값은 2로 계산을 해준다. 즉 동적이다!

    2. 이벤트

      3. 만약 웹페이지에 버튼 하나를 만들고 싶다? 그러면 input 태그를 이용하면 된다. 그리고 그 버튼에 글씨를 쓰고 싶으면 value라는 속성을 이용하면 된다.
      그리고 이 버튼을 눌렀을 때(onclick) 경고창이 나오게 만들고 싶으면 alert값을 넣어주면된다.

      html에서 onclick이라는 속성은 특별한 속성이다. 그 값은 꼭 자바 스크립트가 와야한다. 그리고 onclick이벤트가 발생할 땐 자바 스크립트 문법에 따라서 그 값을 처리하게 된다.

      여기서 말하는 이벤트라는 것은 웹페이지에서 일어나는 일련의 사건을 나타내는 것인데, 위에 예 처럼 버튼을 누르는 이벤트가 있을 수도 있고, 텍스트를 바꾸는 이벤트가 있을 수도 있다. 예를 들면 이런 이벤트도 있을 수 있다.

      이러한 이벤트는 수도 없이 많기 때문에 일일이 다 알고 있을 수가 없다. 그럴 때 이용해야 할 것은 구글에 물어보는 것이다. 예를 들면 JAVASCRIPT KEY DOWN ATTRIBUTE라고 검색해보자.

    3. 콘솔

      4. 지금까지 우리가 자바스크립트를 실행하기 위해서 html파일을 만들었는데, 그렇게 하지 않고도 자바스크립트를 실행시킬 수 있는 방법이 있다. 그 방법이 콘솔을 사용하는것이다. 웹페이지에 들어가서 검사기능을 사용하면 콘솔이라는 창이 있다.

    4. 데이터타입 - 문자열과 숫자

      5. 숫자는 그냥 숫자이다. 그리고 이 숫자를 이용하여 연산을 할 수 있는데, 우리가 초등학교 때 배운 사칙연산을 사용하면 된다.
      그리고 문자열은 ""를 사용하거나 ''를 사용하면 된다. 정말 간단하다.
      문자열을 처리하는 방법 중 하나가 .length라는 것인데, 문자열 안에 문자의 개수를 알려준다. ex 'hello world'.lenght 를 처리하면 11이라는 값이 나온다.
      이와 같이 문자열을 처리하는 기능은 너무 많기 때문에 또 구글의 힘을 빌려보도록 한다. javascript string 이라고 쳐보면 모질라에서 지원하는 웹페이지에 많은 문법들이 나오기 때문에 이것을 참고하면 좋을 것 같다. 예를 들면 hello world를 대문자로 바꾸고 싶다면, 'hello world'.toUpperCase()를 사용하면 된다.
      위에서 배운 개념을 조금 응용하자면, 그냥 숫자 1+1의 결과값은 2가 나온다. 하지만 "1"+"1"을 처리하면 무슨 값이 나올까?
      결과적으로 "11"이라는 값이 나온다. 여기서 우리가 알 수 있는 것은, +라는 기호는 숫자에도 사용할 수 있고 문자열에도 사용할 수 있지만, 결과 값은 조금 다르다는 것이다. 그리고 숫자도 문자열로 처리할 수 있다는 것이다.

    5. 변수와 대입 연산자

      6. 변수는 말그대로 변하는 값이다. 예를들면 x = 1 일때, x가 변수이다. 그리고 대입연산자는 = 라는 기호 인데, 수학에서 = 기호와는 다르게 컴퓨터에서 = 기호는 오른쪽 값을 왼쪽값에 넣어주는 의미를 가진다.
      변수를 사용 할 땐 x=1이라고 써도 되지만 되도록이면 var x=1 이라고 써 주는것이 좋다.
